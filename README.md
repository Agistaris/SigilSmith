# SigilSmith

Release builds for the SigilSmith mod manager.

Latest release: v0.2.0

Downloads:
- https://github.com/Agistaris/SigilSmith/releases

Quick install (Linux):
- AppImage: `chmod +x sigilsmith-0.2.0-x86_64.AppImage` then run it.
- Debian/Ubuntu: `sudo dpkg -i sigilsmith_0.2.0-1_amd64.deb`
- Tarball: `tar -xzf sigilsmith-0.2.0-linux-x86_64.tar.gz` then run `./sigilsmith`.

Verify checksums:
- `sha256sum -c SHA256SUMS.txt`
